Structure of the process $\mathrm{W}^{+}$ + (2,3,4) jets:

               
      /~~~~~~~~~~~~~~~ [0]
  __ /        W+    
 |  |----------------- [1]
 |  |         j1
 |  |----------------- [2]
 |  |         j2
 |__|--------------- [3]
     \       j3
      \---------- [4]
            j4

The numbers indicate the ordering in the event file.
The following cuts are applied:

$$p_{T,j} > 20 GeV, abs(\eta_j) < 6.0, DeltaR_jj > 0.4$$
